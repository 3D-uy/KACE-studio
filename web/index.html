KACE Bootstrap Workflow

  • 1. PACKAGES: Installs core system dependencies.
  • 2. KLIPPER: Installs Klipper host software.
  • 3. MOONRAKER: Installs Moonraker API server.
  • 4. CONFIGS: Prepares system configuration files.
  • 5. MAINSAIL/FLUIDD: Installs the selected UI dashboard.
  • 6. CLIENT CFG: Links appropriate client config.
  • 7. NGINX: Configures web server reverse proxy.
  • 8. SERVICES: Registers systemd start daemons.
  • 9. CROWSNEST: Sets up webcam streaming (if checked).
  • 10. KACE AGENT: Installs KACE agent to close the loop.
